# Node.js Authentication API

A simple authentication API built with **Node.js**, **Express**, **MongoDB**, and **bcrypt**.
This project provides basic user **registration** and **login validation** with password hashing.

---

## Features

- User Registration
  - Validates email and password
  - Prevents duplicate users
  - Hashes passwords using bcrypt
- User Login
  - Validates credentials
  - Secure password comparison
- Proper HTTP status codes
- Clean and minimal structure

---

## Tech Stack

### Frontend

- HTML5
- Vanilla JavaScript (Fetch API)

### Backend

- Node.js
- Express.js
- MongoDB Client
- bcrypt

## Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

2. Start project:

```bash
npm install
npm run dev
```
